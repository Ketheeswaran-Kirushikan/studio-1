# **App Name**: Abibarnman Digital

## Core Features:

- Hero Section Display: Display a full-screen hero section with a circular portrait, intro text, and a 'View My Work' CTA, pulling content details dynamically.
- Vision & Mission Display: Present Loganathan's vision and mission in a structured three-column layout, faithfully replicating the content and formatting.
- Experience Showcase: Display Loganathan's experiences in four boxed cards in a 2x2 grid with appropriate styling and layout to convey experience details.
- Portfolio Project Carousel: Showcase photography and marketing work within the textile/fashion space. Carousel supports description text and product photos, rendered dynamically.
- Contact Form Submission: A contact form allowing potential clients to send messages directly through the website, handling input validation and submission.
- Content Generation: An AI-powered tool that will write a creative description of Loganathan's experience, and create a promotional tweet about a current achievement, providing variations.

## Style Guidelines:

- Primary color: Soft mint green (#C8E6C9) for a clean and fresh background.
- Background color: Very light mint (#F0F4F0), almost white, to maintain minimalist feel, complementing the soft mint green.
- Accent color: Charcoal grey (#37474F) to provide a strong contrast and professional look for text and actionable items.
- Font: 'Poppins', a modern sans-serif for headlines and body text.
- Simple, outline-style social icons for the footer and contact sections.
- Mobile-first, responsive design ensuring optimal viewing experience across all devices. Clean, single-page layout with clear section breaks.
- Subtle fade-in animations on scroll for section reveals, smooth scrolling to enhance user experience.
